💡 Design Tips

  • Use --primary-blue for main interactive elements and headings
  • Apply linear-gradient(135deg, var(--primary-blue), var(--secondary-pink)) for elegant gradients
  • Use --primary-light-blue for hover states to provide feedback
  • Keep #2c3e50 for body text for excellent readability
  • Pair --accent-mint with --primary-blue for fresh accents
  • Use pale pink tags (--secondary-pale-pink) for subtle categorization
  • Avoid pure white backgrounds; use --bg-light (#f8fafb) for softer appearance
  • Test color contrast for accessibility (WCAG AA standard)
