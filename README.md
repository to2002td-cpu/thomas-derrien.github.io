# thomas-derrien.github.io

Personal academic website of Thomas Derrien, PhD student in Computer Science
at Archimedia, IRISA (Univ. Rennes, INSA Rennes, CNRS, Inria).

Live at: https://to2002td-cpu.github.io/thomas-derrien.github.io/

## Structure

- `index.html` — main page (about, news, research, publications, CV, contact)
- `assets/css/style.css` — styles and theme (light/dark)
- `assets/js/main.js` — scroll reveal, active nav highlighting, theme toggle
- `assets/images/` — profile photo and favicon
- `assets/publis/` — publication previews and BibTeX files

## Local preview

```
python3 -m http.server
```

then open `http://localhost:8000`.
